# Data Availability

Due to the large size of the network traffic datasets, they are not included directly in this repository.

## Required Datasets

To reproduce the results, please download the following datasets:

1.  **X-IIoTID**
    *   File: `X-IIoTID dataset.csv`
    *   Source: [Link/DOI]
2.  **Edge-IIoTset**
    *   File: `DNN-EdgeIIoT-dataset.csv` (Selected dataset for ML/DL)
    *   Source: [Link/DOI]
3.  **WUSTL-IIOT-2021**
    *   File: `wustl_iiot_2021.csv`
    *   Source: [Link/DOI]
4.  **TON-IoT**
    *   File: `TON_IoT_Train_Test_Network.csv`
    *   Source: [Link/DOI]

## Directory Structure

Place the downloaded CSV files in this directory or update the paths in the notebook (`notebooks/IIoT_IDS_Analysis.ipynb`).
